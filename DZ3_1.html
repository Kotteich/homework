Блоки и отступы в CSS

1. Ширина и высота (width/height)

Фиксированные размеры: 300×150px
Минимальная ширина: 200px
Максимальная ширина: 80%
Минимальная высота: 50px

2. Внешние отступы (margin)

margin: 20px (все стороны)
margin: 10px 30px (вертикально | горизонтально)
Индивидуальные отступы:
margin-top: 5px
margin-right: 15px
margin-bottom: 25px
margin-left: 35px
margin: 0 auto (центрирование)

3. Внутренние отступы (padding)

padding: 20px (все стороны)
padding: 10px 30px (вертикально | горизонтально)
Индивидуальные отступы:
padding-top: 5px
padding-right: 15px
padding-bottom: 25px
padding-left: 35px

4. Границы (border)

border: 3px solid #3498db
border: 2px dashed #e74c3c
border: 1px dotted #2ecc71
Разные границы для каждой стороны

5. Скругление углов (border-radius)

border-radius: 5px
border-radius: 15px
border-radius: 30px
Круг (50%)
Неравномерное скругление (15px 5px 30px 50px)

6. Тень блока (box-shadow)

Простая тень (3px 3px 5px)
Тень с размытием (0 0 15px)
Несколько теней
Внутренняя тень (inset)

7. Тип отображения (display)

display: block
span с display: block

display: inline еще один inline

display: inline-block еще один inline-block
Этот блок не отображается (display: none)

8. Видимость (visibility)

Видимый элемент
Скрытый элемент (visibility: hidden)
Видимый элемент (обратите внимание на место, которое занимает скрытый элемент)

9. Переполнение (overflow)

overflow: visible

Этот текст выходит за границы блока, потому что он слишком длинный для заданных размеров контейнера.

overflow: hidden

Этот текст обрезается, потому что он слишком длинный для заданных размеров контейнера.

overflow: scroll

Этот текст можно прокрутить, потому что он слишком длинный для заданных размеров контейнера.

overflow: auto

Этот текст можно прокрутить только при необходимости.

10. Box-sizing

box-sizing: content-box (по умолчанию)

Ширина: 200px + padding: 20px + border: 5px = 250px

box-sizing: border-box

Ширина: 200px (включая padding и border)
