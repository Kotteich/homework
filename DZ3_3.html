Фоны в CSS - Комбинированный пример

1. Сплошной цвет фона (background-color)

Элемент с заданным сплошным цветом фона.

background-color: #3498db;
background-color: #3498db
background-color: rgba(231, 76, 60, 0.7)
background-color: hsl(120, 76%, 50%)

2. Фоновое изображение (background-image)

Элемент с фоновым изображением, которое масштабируется и центрируется.

background-image: url('image.jpg');
background-size: cover;
background-position: center;

3. Градиентный фон (linear-gradient)

Элемент с линейным градиентом в качестве фона.

background: linear-gradient(135deg, #ff7e5f, #feb47b);
linear-gradient(to right, #3498db, #2ecc71)
radial-gradient(circle, #3498db, #2c3e50)

4. Повторение фона (background-repeat)

Элемент с повторяющимся фоновым изображением (паттерн).

background-image: url('pattern.png');
background-repeat: repeat;

Также доступны значения: repeat-x, repeat-y, no-repeat, space, round

5. Позиционирование фона (background-position)

Элемент с фоновым изображением, позиционированным в правом нижнем углу.

background-image: url('logo.png');
background-repeat: no-repeat;
background-position: right bottom;

6. Размер фона (background-size)

Элемент с фоновым изображением, которое масштабируется с помощью background-size: cover (заполняет весь элемент).

background-size: cover;

background-size: contain

Элемент с фоновым изображением, которое масштабируется с помощью background-size: contain (вписывается в элемент).

background-size: contain;
background-size: contain

7. Фиксация фона (background-attachment)

Элемент с фиксированным фоном, который остается на месте при прокрутке.

background-image: url('large-bg.jpg');
background-attachment: fixed;
background-size: cover;

8. Обрезка фона (background-clip)

Определяет, как фон распространяется под границами и отступами.

border-box (по умолчанию)

Фон распространяется под границами.

background-clip: border-box;

padding-box

Фон обрезается по внутреннему краю границы.

background-clip: padding-box;

content-box

Фон обрезается по контенту.

background-clip: content-box;

9. Множественные фоны

Элемент с несколькими фоновыми изображениями и градиентом.

background:
  url('image1.png') left top no-repeat,
  url('image2.png') right top no-repeat,
  url('image3.png') left bottom no-repeat,
  url('image4.png') right bottom no-repeat,
  linear-gradient(135deg, rgba(255, 255, 255, 0.9), rgba(236, 240, 241, 0.9));

10. Фон для текста (background-clip: text)

Применение фона только к тексту (вебкит-свойство).

Текст с градиентным фоном
background: linear-gradient(45deg, #ff7e5f, #feb47b, #2ecc71, #3498db);
-webkit-background-clip: text;
background-clip: text;
color: transparent;
